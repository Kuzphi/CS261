	Let's say the key set is $\{k * 2^{16}| 0 \le k < 2^{16}\}$. For any randomly choosen a,b:
	\begin{flalign*}
		&H(x) = (ax + b) \mod 2^{16}&\\
		&= (ak * 2^{16}) \mod 2^{16} + b \mod 2^{16}\\
		&= b
	\end{flalign*}
	Then all the key would fall in same slot $b$.
